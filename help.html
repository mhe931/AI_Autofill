AI Form Filler - Documentation

This extension is designed for Daniel Ebrahimzadeh to automate the Recognition of Prior Learning (RPL) process at the University of Vaasa[cite: 17, 24, 25].

Automated Mapping: Instantly fills academic justifications based on Master's level research and professional Data Manager experience[cite: 42, 63, 113].

3-Button Analyze-Generate-Inject Workflow [cite: 2026-03-12]

  1. Step 1: HTML Extraction. Open the target form (e.g., Webropol) and click "Extract Form HTML" in the extension popup. This scrapes the form and prepends the system instruction automatically to your clipboard.
  2. Step 2: Contextual LLM Interaction. The extract script will automatically inject historical tabular context (like specific paths to "Social security translate.pdf" or historical answers) into your prompt. Paste this securely into your LLM.
  3. Step 3: Dynamic Injection. Copy the resulting JSON output from the LLM. Go back to the form page and click "Execute Injection" in the extension. The extension will automatically read the JSON from your clipboard (or the text box) and intelligently fill out the fields via fuzzy matching [cite: 85, 95].
  4. Review the data and manually upload the required PDF attachments [cite: 2026-01-24].

Data Warehouse & Persistence

The extension allows you to build a local dataset. Use the Data Management panel:

  • Capture Current Form Data: After successfully completing a form, click this to save all values (and any manually-typed local file paths) into a tabular relation tracking Labels, URLs, and Time.
  • View Data Table: Generate an immediate HTML visualization of all saved mappings.
  • Export / Import: Download your entire relational mapping schema as a .json file to backup or migrate between workstations [cite: 2026-03-12].

Data Privacy

All data is stored locally within chrome.storage.local. No information is transmitted to external servers [cite: 2026-01-24].
