AI Form Filler - Documentation

A general-purpose automation suite for intelligent form mapping and dynamic data injection.

Developed by Daniel Ebrahimzadeh.

How It Works: Configure your professional profile once in the User Profile section. The extension merges your profile with the target form HTML into a single LLM-ready prompt. You paste it into any AI (Gemini, ChatGPT, Claude, etc.), copy the JSON answer back, and the extension fills the form automatically.

Getting Started

  1. Set Up Your Profile: Open the extension popup and click "User Profile" to expand the profile section. Paste your resume, professional bio, or any context you want the LLM to use when filling forms. Click "Save Profile". This is stored locally and persists across sessions.
  2. Navigate to a Form: Go to any web form you want to fill (job application, registration, etc.).
  3. Follow the 3-step workflow below.

3-Step Workflow

  1. 1 → Extract (Profile + Form HTML)
    Click "Extract Form HTML". The extension scrapes all form elements, merges them with your saved profile and any historical data from the Local Warehouse, and produces a complete LLM prompt. This prompt appears in the textarea and is copied to your clipboard.
  2. 2 → Paste to AI
    Open an LLM interface. Click "Paste to LLM". The extension reads directly from its textarea and injects the prompt into the AI chat input. No clipboard dependency at this step.
  3. 3 → Paste AI JSON & Execute
    Copy the LLM JSON response. Paste it into the extension textarea (replacing the prompt). Click "Execute Injection". The extension strips any ```json markdown fences, validates JSON, then maps keys to form fields using multi-signal matching (id, name, label, placeholder, container text).

User Profile

The User Profile section in the popup stores your personal context (resume, bio, skills, publications, etc.) in chrome.storage.local. This text is injected verbatim into the ### USER PROFILE section of every generated prompt.

  • Paste any format: plain text, structured sections, or bullet points.
  • The more detail you provide, the better the LLM can fill specialized fields.
  • Profile data never leaves your browser. It is stored locally and included only in prompts you manually send to an LLM.
  • Click "Clear Profile" to remove all stored profile data.

Textarea Controls

  • Clear: Empties the textarea and clears the persisted content.
  • Copy: Copies the current textarea content to the system clipboard.
  • Paste: Reads the system clipboard and replaces the textarea content.

Background Persistence: The textarea content is automatically saved to chrome.storage.local on every keystroke. Content is restored when you reopen the popup.

Data Warehouse & Persistence

All data stays local via chrome.storage.local:

  • Capture Current Form Data: Scrapes all filled fields into timestamped records for future prompt augmentation.
  • View Data Table: Opens a formatted HTML table of all captured context.
  • Export / Import: Full JSON backup and restore for cross-machine portability.

Troubleshooting

Clipboard Permission Errors

If extraction fails to copy to clipboard:

  1. Click the Lock Icon in the URL bar.
  2. Select Site Settings.
  3. Set Clipboard to Allow.
  4. Reload the page and retry.

Note: Even if clipboard copy fails, the prompt still loads into the textarea. Copy it manually.

Connection Errors

If you see "Receiving end does not exist" or "Reload the target page to reconnect":

  1. The extension listener on the target page was disconnected.
  2. Refresh the target page (F5 or Ctrl+R) to restart the listener.
  3. Try again. The extension also attempts automatic re-injection on first failure.

URL Support Limitation

The extension cannot inject into chrome:// or edge:// pages. Navigate to an http:// or https:// target.

Best Practices

  • Configure your profile first. A detailed profile produces much better LLM responses.
  • Active Tab Focus: Click into the target tab before pressing Extract.
  • LLM Input Focus: Click inside the LLM chat box before pressing Paste.
  • JSON Fences: The extension auto-strips ```json wrappers.
  • Capture Often: Run Capture after each form to build historical context.
  • Use Clear: Clear the textarea before pasting new JSON to avoid mixing data.
