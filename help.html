AI Form Filler - Documentation

A general-purpose toolkit developed by Daniel Ebrahimzadeh.

Overview: This extension empowers users to seamlessly transfer full form contexts to an external LLM, augment prompts with historical local data, and dynamically inject mapped responses back into the DOM using intelligent fuzzy matching algorithms.

3-Button Analyze-Generate-Inject Workflow

  1. Step 1: Ensure Site Permissions allow Clipboard access. If an action fails, check your browser's site settings.
  2. Step 2: Use 'Extract HTML' for target context. Open your target form. Click "Extract Form HTML" to securely write the standardized architecture and contextual arrays to your clipboard.
  3. Step 3: Process via AI and execute 'Dynamic Injection'. Feed the clipboard query to external LLMs, copy the JSON response, and trigger 'Execute Injection' inside the target tab for automated, heuristic mapping.

Data Warehouse & Persistence

We advocate a strictly "Local First" architecture:

  • Capture Current Form Data: After finishing a form logically, press Capture to safely register those text responses or exact local file paths into a persistent database.
  • View Data Table: Generate an immediate HTML visualization of all saved context mappings.
  • Export / Import: Use JSON blobs for system-agnostic migrations across workstations.

Troubleshooting

Resolving Clipboard Access Errors

Browsers restrict clipboard read/write access to prevent abuse. If you see a "Permission Denied" alert:

  1. Click the Lock Icon in your URL bar next to the domain name.
  2. Select Site Settings.
  3. Scroll down to Clipboard and change it from 'Ask' (or 'Block') to 'Allow'.
  4. Reload the page and try again.

Best Practices

  • Active Tab Focus: Always ensure the target tab is active (clicked into) before pressing Extract Form HTML. This maintains clipboard write access in browsers that restrict background clipboard operations.
  • LLM Paste: Before clicking Paste to LLM, click once directly inside the LLM chat input field to ensure the target element is focused and detectable by the injection engine.
  • JSON Integrity: Validate LLM-generated JSON before pasting into the textarea. Malformed JSON silently blocks the injection pipeline.
  • Data Capture Cadence: Run Capture Current Form Data after each major form completion to maintain a high-fidelity local context for future prompt augmentation.
