AI Form Filler - Documentation

An intelligent automation suite developed by Daniel Ebrahimzadeh.

Overview: This extension extracts form HTML, merges it with your professional profile and historical context, and copies the combined prompt to the clipboard. You paste it into any LLM, then inject the AI-generated JSON response back into the form for automated filling.

3-Step Workflow

  1. Extract (Form HTML + Profile)
    Navigate to your target form. Click "Extract Form HTML". The extension scrapes all <form> elements (or the full DOM) and prepends Daniel Ebrahimzadeh's professional profile, the system instructions for the LLM, and any historical context from the Local Data Warehouse. The complete prompt is copied to your clipboard. You'll see "HTML + Profile Copied!" in the overlay.
  2. Paste to AI
    Navigate to an LLM interface (Gemini, ChatGPT, Claude, etc.). Click "Paste to LLM". The extension reads your clipboard and injects the full prompt into the active chat input field using execCommand('insertText') for maximum framework compatibility. If the page is detected as a known LLM domain, the paste proceeds directly; otherwise, the extension warns but still attempts injection.
  3. Paste AI JSON & Execute Injection
    Copy the LLM's JSON response. Paste it into the extension's textarea (or leave empty to read from clipboard). Click "Execute Injection". The extension strips markdown code fences, validates JSON, and maps keys to form fields using a multi-signal fuzzy matcher (id, name, label, placeholder, container text). Each matched field is filled and dispatches input and change events.

Data Warehouse & Persistence

All data stays local via chrome.storage.local:

  • Capture Current Form Data: Scrapes all filled fields into a timestamped record set.
  • View Data Table: Opens a formatted HTML table of all captured context.
  • Export / Import: Full JSON backup and restore for cross-machine portability.

Troubleshooting

Clipboard Permission Errors

If extraction or pasting fails with "Permission Denied":

  1. Click the Lock Icon in the URL bar.
  2. Select Site Settings.
  3. Set Clipboard to Allow.
  4. Reload the page and retry.

URL Support Limitation

The extension cannot inject scripts into internal browser pages (chrome://, edge://). Navigate to an http:// or https:// page.

Best Practices

  • Active Tab Focus: Click into the target tab before pressing Extract to maintain clipboard write access.
  • LLM Input Focus: Click inside the LLM chat input box before pressing Paste to ensure element detection.
  • JSON Validation: If the LLM wraps output in ```json fences, the extension strips them automatically. But verify structure if injection fails.
  • Capture Often: Run Capture after each form completion to build historical context for future prompts.
